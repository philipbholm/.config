# Learnings from PR #1286

Replaced AWS Secrets Manager credential loading with environment variable injection via Docker Compose, and removed the associated SDK dependencies.

---

## 1. Use Schema Validation Instead of Manual Runtime Checks

When a schema validation library (like Zod) is already parsing and validating environment variables at startup, writing additional manual `if (!value) throw` guard clauses is redundant. The schema already guarantees the value is present and correctly typed by the time the code runs, so manual checks add noise without safety.

**Guideline:** If environment variables are validated through a schema (e.g., Zod), do not add separate manual existence checks for the same variables.

---

## 2. Consolidate Environment Validation Into a Single Layer

The PR originally had a dedicated `loadCodelistCredentials` function that re-checked env vars that were already defined as required in the Zod schema. Having validation split across multiple layers (schema + custom function) makes it unclear which layer is the source of truth and leads to redundant code that drifts over time.

```typescript
// Redundant: Zod schema already requires these fields
function loadCodelistCredentials(environment: Environment) {
    if (!environment.CODELIST_CLIENT_CREDENTIALS_ID) {
        throw new Error("CODELIST_CLIENT_CREDENTIALS_ID is not set");
    }
    // ...
}
```

**Guideline:** Validate each configuration value in exactly one place -- prefer the schema layer -- and trust it downstream.

---

## 3. Remove Unused Dependencies When Changing Architecture

Switching from runtime secret loading (AWS Secrets Manager SDK) to environment variable injection allowed the removal of `@aws-sdk/client-secrets-manager` and `@aws-sdk/credential-providers`, eliminating ~600 lines of lockfile entries. Leaving unused SDK dependencies inflates bundle size, increases attack surface, and slows installs.

**Guideline:** When changing how configuration or secrets are loaded, audit and remove any SDK or library dependencies that are no longer needed.
